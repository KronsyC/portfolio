# About Me

- My languages of choice are Rust, C++ and Python with a hint of Typescript.

- My current areas of interest with regard to programming are Distributed Computing and Compilers.

- When I was 16, I built a compiler for my own programming language [Quinoa](https://github.com/KronsyC/quinoa). Utilizing LLVM as its code-emitting backend.

- When I was 15, I built a web framework in Typescript from scratch, with components including my own JSON schema validation library, my own router library, and my own library to unify the Node.js Http1 and Http2 APIs.
